# Security Policy

## Описание (Description)
Библиотека **NecroUI** использует закрытый метод загрузки (loadstring) из публичного репозитория. Локальные конфиги игроков (JSON) хранятся исключительно на их устройствах через стандартные средства движка Roblox (writefile/readfile в `workspace/` или изолированного локального хранилища экзекутора). 

Библиотека не отправляет никаких телеметрических или иных данных на сторонние сервера (исключение: загрузка самой себя по GitHub Raw URL). Выполнение вашего кода внутри `Callback` функций полностью изолировано от внутренней логики библиотеки.

## Выпуск обновлений (Update Methodology)
Поскольку загрузка осуществляется при каждом запуске через GitHub Raw (обходя кэш через `?t=tick()`), **пользователь автоматически получает самую актуальную и безопасную версию** при каждом инжекте скрипта без необходимости ручного обновления.

## Как сообщить об уязвимости
Откройте **Issue** в репозитории на GitHub с детальным описанием сбоя или уязвимости. Пожалуйста, не публикуйте эксплоиты публично до выхода патча!
